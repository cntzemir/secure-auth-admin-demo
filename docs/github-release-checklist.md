# GitHub release checklist

## Repository presentation
- [ ] Repository is public
- [ ] Repository description is filled in
- [ ] Relevant topics are added
- [ ] Project is pinned on the GitHub profile

## Code hygiene
- [ ] `.env` is ignored
- [ ] runtime data files are ignored
- [ ] `node_modules/` is ignored
- [ ] screenshots exist in `screenshots/`
- [ ] README image links render correctly on GitHub

## Product/demo sanity check
- [ ] Register flow works
- [ ] Login / logout works
- [ ] Forgot-password mock flow works
- [ ] User dashboard, profile, and activity pages work
- [ ] Admin dashboard, users, logs, and locked accounts pages work
- [ ] Failed login and lockout behavior can be demonstrated
- [ ] Audit logs record expected auth/admin events

## Final GitHub polish
- [ ] README is the final version
- [ ] Default admin credentials are documented
- [ ] Docs links in README work
- [ ] Initial commit history looks clean enough for public viewing
